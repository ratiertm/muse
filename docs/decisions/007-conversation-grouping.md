# ADR-007: 대화 목록 캐릭터별 그룹핑

- **Status**: Accepted
- **Date**: 2026-03-22
- **Context**: 같은 캐릭터와 여러 번 대화하면 목록에 중복 항목이 쌓여서 스크롤이 길어지고 찾기 어려움.
- **Decision**: 1:1 대화는 같은 캐릭터끼리 그룹핑하여 최신 1건만 표시. 그룹 채팅은 개별 표시. "캐릭터 대화" / "시나리오 대화" 섹션으로 분류.
- **Alternatives considered**:
  - 모든 대화 개별 표시 → 목록 비대화
  - 캐릭터별 폴더 구조 → 2단계 탐색 필요, 오버엔지니어링
- **Consequences**:
  - 목록이 깔끔해짐
  - 과거 대화 접근이 한 단계 더 필요 (그룹 탭 → 히스토리)
  - autoDispose provider로 탭 전환 시 자동 갱신
