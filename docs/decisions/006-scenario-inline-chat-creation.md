# ADR-006: 시나리오 상세 바텀시트에서 채팅 직접 생성

- **Status**: Accepted
- **Date**: 2026-03-22
- **Context**: 기존에는 시나리오 선택 → 별도 "그룹 채팅 만들기" 화면으로 이동 → 다시 시나리오/캐릭터 선택. 동선이 길고 중복 선택이 발생.
- **Decision**: 시나리오 카드 탭 → 바텀시트에 상세 + 채팅방명/페르소나/캐릭터 선택 폼 통합 → "시작하기"로 바로 생성
- **Alternatives considered**:
  - 별도 화면 유지 → 동선 3단계, UX 불편
  - 다이얼로그 → 공간 부족, 캐릭터 선택 어려움
- **Consequences**:
  - 시나리오 → 채팅 시작이 2탭으로 단축
  - 바텀시트가 길어질 수 있음 (캐릭터 많을 때)
  - 그룹 채팅 만들기 화면은 FAB에서 직접 접근 시에만 사용
