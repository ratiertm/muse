# App Icon

Place your app icons here:
- `app_icon.png` - 1024x1024 PNG (main icon)
- `app_icon_fg.png` - 1024x1024 PNG (foreground for adaptive icon, optional)

## Generate Icons

After adding your icon images, run:
```bash
flutter pub run flutter_launcher_icons
```

## Temporary Icon

For now, we'll use Flutter's default icon. To create a simple Muse logo:

1. Create a 1024x1024 PNG with dark background
2. Add a large "M" letter (for Muse) in white/accent color
3. Name it `app_icon.png`
4. Run the icon generator

Or use an online tool like:
- https://icon.kitchen/
- https://appicon.co/
