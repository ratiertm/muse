# Muse APK Installation Guide

이 가이드는 Muse 앱을 안드로이드 기기에 설치하는 방법을 안내합니다.

---

## 빌드 (개발자용)

### 1. 사전 준비
- Flutter SDK 설치 필요 (https://flutter.dev)
- Android Studio 또는 VS Code

### 2. APK 빌드
```bash
cd frontend
./build_apk.sh prod
```

**커스텀 서버 URL 사용:**
```bash
./build_apk.sh prod http://your-server-ip:8000
```

빌드된 APK 위치: `build/app/outputs/flutter-apk/app-release.apk`

---

## 설치 (사용자용)

### 방법 1: ADB로 설치 (PC 연결)

1. **개발자 옵션 활성화:**
   - 설정 → 휴대전화 정보 → 소프트웨어 정보
   - "빌드 번호" 7번 연속 탭
   - "개발자가 되었습니다" 메시지 확인

2. **USB 디버깅 활성화:**
   - 설정 → 개발자 옵션 → USB 디버깅 켜기

3. **USB로 PC와 연결**

4. **ADB로 설치:**
   ```bash
   adb install app-release.apk
   ```

### 방법 2: 직접 설치 (권장)

1. **APK 파일을 휴대폰으로 전송:**
   - 카카오톡, 이메일, USB 케이블 등 이용
   - 또는 Google Drive/Dropbox에 업로드 후 휴대폰에서 다운로드

2. **출처를 알 수 없는 앱 설치 허용:**
   - Android 8.0 이상:
     - APK 파일 탭 → "이 출처에서 허용" 활성화
   - Android 7.x 이하:
     - 설정 → 보안 → 출처를 알 수 없는 앱 허용

3. **APK 파일 실행:**
   - 파일 관리자 또는 다운로드 앱에서 APK 파일 탭
   - "설치" 버튼 클릭

4. **설치 완료:**
   - "앱 설치됨" 메시지 확인
   - "열기" 탭하여 실행

---

## 첫 실행

1. **앱 실행**
   - 홈 화면에서 Muse 아이콘 탭

2. **프로필 선택**
   - MB 또는 딸 선택

3. **PIN 입력**
   - 기본 PIN: `1234` (향후 변경 가능)

4. **서버 연결 확인**
   - 캐릭터 목록이 보이면 정상
   - "서버에 연결할 수 없습니다" 에러 시 → 설정에서 서버 URL 확인

---

## 서버 설정 (필요시)

앱 실행 후:
1. 왼쪽 상단 메뉴 (☰) → 설정
2. "서버 URL" 항목 탭
3. 올바른 서버 주소 입력:
   - 로컬 네트워크: `http://192.168.x.x:8000`
   - Oracle Cloud: `http://your-server-ip:8000`
   - 도메인: `https://your-domain.com`
4. 저장 후 앱 재시작

---

## 권한 설정

Muse는 다음 권한을 요청합니다:

- **인터넷 접근** ✓ (필수)
  - 서버와 통신, LLM API 호출
- **저장소 접근** (선택)
  - 프로필 사진 업로드 시

모든 권한은 필요할 때만 요청됩니다.

## 업데이트

새 버전이 출시되면:

1. 새 APK 다운로드
2. 기존 앱 위에 덮어쓰기 설치
3. 데이터는 유지됩니다 (서버 저장)

---

## 삭제

앱 제거:
- 설정 → 앱 → Muse → 제거

**주의:** 
- 로컬 데이터만 삭제됩니다
- 서버에 저장된 캐릭터/대화는 유지됩니다
- 재설치 후 같은 프로필로 로그인하면 복구됩니다

---

## 문제 해결

### "앱을 설치할 수 없습니다"
- 이전 버전 완전 삭제 후 재시도
- 저장 공간 확인 (최소 100MB 필요)
- APK 파일 재다운로드

### "서버에 연결할 수 없습니다"
- WiFi/모바일 데이터 연결 확인
- 서버 URL이 올바른지 확인
- 서버가 실행 중인지 확인
- 방화벽 설정 확인

### "로그인이 만료되었습니다"
- 프로필 선택 화면으로 돌아가서 재로그인
- PIN 재입력

### 앱이 느려요
- 대화 기록이 많으면 속도 저하 가능
- 오래된 대화 삭제 고려
- 백그라운드 앱 종료

### 아바타가 안 보여요
- 인터넷 연결 확인
- 이미지 로딩 시간 필요 (최대 30초)
- 앱 재시작

## 시스템 요구사항

- **OS:** Android 7.0 (API 24) 이상
- **RAM:** 최소 2GB (권장 4GB)
- **저장 공간:** 최소 100MB
- **인터넷:** WiFi 또는 모바일 데이터 (LLM API 호출)
