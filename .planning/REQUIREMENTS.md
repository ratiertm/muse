# REQUIREMENTS.md — Muse v1

## v1 Requirements

### REQ-01: 캐릭터 생성
- 이름, 성격, 말투, 배경스토리, 시나리오, 첫 메시지, 예시 대화
- 태그/카테고리 분류
- 캐릭터 카드 구조 (SillyTavern V2 스펙 참고)

### REQ-02: 원작 기반 캐릭터 자동 생성
- 작품명 + 캐릭터명 입력 → AI가 성격/말투/배경 자동 채움
- 유저가 수정 가능

### REQ-03: 1:1 채팅
- 캐릭터와 실시간 대화
- SSE 스트리밍 응답 (타이핑 효과)
- 메시지 재생성 (swipe)
- 메시지 편집/삭제
- 대화 기록 저장 및 이어하기

### REQ-04: God Agent 시스템
- 전지적 오케스트레이터
- World State 관리 (시간축, 장소, 이벤트)
- Knowledge Graph (캐릭터별 아는 것/모르는 것)
- Private State (캐릭터 감정, 비밀)
- 대화 전 캐릭터 브리핑 생성
- 대화 후 세계 상태 자동 업데이트
- 이벤트 추출 및 지식 전파

### REQ-05: 그룹 채팅
- 여러 캐릭터와 동시 대화
- God Agent가 턴 순서 관리
- 캐릭터 간 지식 차이 반영

### REQ-06: 멀티 프로필
- 유저별 계정 분리 (MB + 딸)
- 각자 캐릭터/대화/시나리오 독립
- 간단한 로그인 (PIN or 프로필 선택)

### REQ-07: 애니풍 캐릭터 아바타 AI 생성
- 캐릭터 생성 시 설정 기반 자동 이미지 생성
- 애니/만화 스타일
- 직접 이미지 업로드도 가능

### REQ-08: 시나리오 관리
- 시나리오 생성 (세계관, 배경, 규칙)
- 시나리오에 캐릭터 배치
- 같은 캐릭터를 다른 시나리오에서 재사용

### REQ-09: 유저 페르소나
- 유저 자신의 캐릭터 설정 (이름, 외모, 성격)
- 시나리오별 다른 페르소나 사용 가능

### REQ-10: LLM 멀티 모델
- GPT-4o-mini (God Agent, 빠른 응답)
- Claude Sonnet (캐릭터 응답, 고품질)
- 캐릭터별 모델 선택 가능
